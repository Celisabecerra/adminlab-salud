🔬 AdminLab en GitHub Pages GUÍA RÁPIDA

Paso a paso para publicar el curso y obtener el link que pegas en Morfeo · ~5 minutos

🎯 Al terminar esta guía tendrás
celisabecerra.github.io/adminlab-salud/

Un link permanente, gratuito y sin expiración que pegas en Morfeo para que tus estudiantes accedan al curso completo de las 4 etapas del proceso administrativo.

📄

Archivo que necesitas tener descargado en tu computador

El curso AdminLab que ya construimos juntas. Descárgalo desde esta sesión de Claude si aún no lo tienes:

AdminLab_ProcesoAdministrativo_Bacteriologia.html
1

Inicia sesión en GitHub

Abre github.com en tu navegador e inicia sesión con tu cuenta celisabecerra (la misma de Habilidades Blandas).

💡 Si no recuerdas la contraseña, usa "Forgot password" con tu correo institucional.
2

Crear el repositorio nuevo

En la esquina superior derecha, clic en el ícono "+" → "New repository".

Completa el formulario así:

• Repository name: adminlab-salud
• Description: Curso AdminLab — Proceso Administrativo en Salud
• Selecciona Public (obligatorio para GitHub Pages gratuito)
• Marca "Add a README file"
• Clic en "Create repository" (botón verde)

3

Subir el archivo HTML

Dentro del repositorio recién creado, clic en "Add file" → "Upload files".

Arrastra el archivo AdminLab_ProcesoAdministrativo_Bacteriologia.html a la zona de carga.

⚠️ Importante: Antes de guardar, renombra el archivo. Haz clic en el nombre del archivo subido y cámbialo a exactamente: index.html — esto es lo que permite que GitHub Pages lo muestre como página principal.

En el campo de commit escribe: Agregar curso AdminLab → clic en "Commit changes" (botón verde).

4

Activar GitHub Pages

En el repositorio, clic en "Settings" (ícono engranaje, parte superior).

En el menú lateral izquierdo, busca y clic en "Pages".

En la sección "Branch":
• Cambia de None a main
• Deja la carpeta como / (root)
• Clic en "Save"

✅ GitHub mostrará el mensaje: "Your site is live at..." — esto puede tardar 1 a 2 minutos en activarse.
5

Verificar que funciona

Espera 2 minutos y abre en el navegador:

https://celisabecerra.github.io/adminlab-salud/

Deberías ver el curso AdminLab completo con los 4 módulos, el sistema de XP y los retos. ✅

💡 Si ves error 404, espera 2 minutos más y recarga. GitHub Pages tarda un momento en publicar por primera vez.

🎓 Paso 6 — Agregar el link en Morfeo

A

Entra a tu curso de Administración en Salud en Morfeo → clic en Activar edición.

B

En la sección del módulo correspondiente → "Agregar actividad o recurso".

C

Selecciona "URL" (no "Archivo" — usa URL para links externos).

D

En el campo "URL externa" pega:
https://celisabecerra.github.io/adminlab-salud/

E

En "Apariencia" → "Mostrar" → selecciona "En ventana emergente" o "Incrustado" para que se abra dentro de Morfeo sin salir de la plataforma.

F

Clic en "Guardar y regresar al curso". ¡Listo! Los estudiantes verán el enlace y al hacer clic accederán al curso completo. 🎉

⚡ Recuerda sobre la vigencia y los videos

El link de GitHub Pages es permanente y gratuito — no expira. Si algún video no carga, usa el panel de búsqueda de videos que está al inicio del curso (los botones azules/verdes) para encontrar el video correcto en YouTube, copiar el ID y actualizarlo en el archivo index.html del repositorio directamente desde GitHub (clic en el lápiz ✏️).
